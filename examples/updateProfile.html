This is a really simple script that shows you all the fields you can update on your profile.
Grabbing the token from the login.html example and replacing the in each of the forms will let you update them.
Each field is case sensitive as spelt by the bolded text.

All of these use the http://ws.anomo.com/v210/index.php/webservice/user/update/ endpoint. Use the
updateAvatar.html example to update the avatars. Location is a different endpoint that I'll add an example for. Other fields in the response are not modifiable, like IsAdmin or Credits.

UserName. This has to be uppercase A through Z and 0 through 9. They don't allowe special
characters anymore and filter for swears in names. It used to not care how long you made your name but stopped
exactly the same because of the display characters visible but the ends were off by a couple characters that no one
would ever see. You can only change your username once ever calendar
month. That means if you changed your username on the 15th you won't be able to change it until the 16th.


Password can be anything you want as long as it's in MD5 encrypted format. I guess if you're just using
the api that you could just use plain text passwords but note that the app itself encrypts it to MD5 when it
sends so it wouldn't match if you logged in using the app.


BirthDate formatted with YYYY-MM-DD you can set it to any date and it will calculate based on that year.


AboutMe limited to about 175 characters you can put any text in here that you'd like. You can use
\n
to add carriage returns.


Gender 1 represents female, 2 represents male, and 0 doesn't change your gender but returns an
error on the avatar screen, because the server returns {"code":"INVALID_REQUEST"}.


FacebookID This used to show the users FacebookID if they had verified with FB. Now it shows a 1 if it is.
You should still be able to use this field to update your FBid on file so if you want to move your fb login to
another account you'd update this field. This can come in handy for verifying an account with FB and just move
it around as you need. Remember to always have a password set to the account.


CoverPicture this is based on a file uploaded to the s3 bucket. You can upload to it using the revealUpload
example and use that image file to update this field. I'm not sure of the size specifications since i didn't
really use this much.


The resulting response from Anomo should be JSON formated data. You can copy the text and paste it into jsonviewer.stack.hu for viewing in a prettier format.
