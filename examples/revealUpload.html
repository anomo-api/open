This is a really simple way of uploading images for the purpose of changing your avatar
Replace the in the form url with the token generated in the Login.html example page.
The result will be in JSON with a link to the image similar to this https:\/\/anomo-production1.s3.amazonaws.com\/upload\/
The file name at the end of that url is what needs to be updated for the Avatar. Save it for the avatar.html example page.
