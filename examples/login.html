This is a really lazy and simple example to show logging into Anomo using username and password.
The login endpoint expects UserName and Password to be passed to it in a POST request.
Username is not case sensitive however the password is hashed using MD5 encryption.
If you don't know the MD5 hash for your password, which I suspect most won't, you can use
http://www.miraclesalad.com/webtools/md5.php to encrypt it to use in this form.
