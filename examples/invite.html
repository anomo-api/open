This is a really lazy and simple example of how to use the invite function to increase your user points.
Since they don't validate numbers, for international reasons, you can pass any numeric value to the invite service to count. The invite.php generates numbers based on an area code you provide, to look more legit, but anything will work.
Invite.php calculates the number of points the user has and subtracts it from 130,000 to determine how many points the
user is missing from getting a Fan Page. It then generates that many phone numbers, rounded up, to invite.
You can have any number of consecutive entries as long as they're separated by a semi colon ";".
As per usual, replace with the token generated when you used login.html.
