Use this endpoint to login using the users's username and password. Unless they've already set one and registered with FB then they won't have a password. You can optionally use the Password or EncPassword depending on what you or the user prefers to use.

Notable: Tokens are valid until the next time the user credentials are used to log into the account.

Method:Post

Variable Description
FacebookID Username of of the user intending to login.
Password The users password in clear text. Option 1 for password.
EncPassword Users password in MD5 encryption. Option 2 for password.
